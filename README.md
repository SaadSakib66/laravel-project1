# Laravel Project

## Project Overview
Portfolio website.

## Features
- Home
- Resume
- Project
- Contact

## Installation Process
1. Clone the repository
2. Run composer install
3. Copy .env.example to .env
4. Run php artisan key:generate
5. Setup database
6. Run php artisan migrate
